Introducing Unhosted Accounts

How Open Web Fails to Solve This

  • Open Web is great.

  • OpenId, Webfinger, Rss, ActivityStreams, Salmon, Xmpp, OExchange.

  • But standards will usually lag about 2 years behind on features. And it's a lot of work to coordinate, standardize, then implement all of this each time.

  • Result: It half-works. I can follow a Diaspora user on Identica. But not send a private message cross-site.
