Introducing Unhosted Accounts

SyncStorage API (draft)

  • Connection:
    syncStorage = new SyncStorage();
    syncStorage.linkAccount('user@host');
    syncStorage.getWriteAccess(); //callback to same page (assume it's a SPA)

  • Commands:
    syncStorage.fetchItem('key')
    syncStorage.setItem('key', 'val')
    syncStorage.removeItem('key')

  • Feedback:
    syncStorage.onStatus = function('offline'/'readonly'/'sending'/'receiving'/'ready', 'user@host')
    syncStorage.onItem = function('key', 'val')
