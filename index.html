Separating web apps from data storage

The web is not as open as it used to be: monopoly platforms formed new proprietary layers on top of it. But we create a better architecture for the web. We break the package deal »you get our app, we get your data« with remoteStorage, a cross-origin data storage protocol separating application servers from data storage.

Unhosted is an open non-profit project. Participate on code, mailing list and chat room. Follow @unhosted or find more links in the footer of this page.

web apps

Applications are pure client-side Javascript, HTML & CSS.
That way they are not bound to a specific host. Try a demo app (user address: 'test@yourremotestorage.com', then user: 'test', password 'unhosted').
